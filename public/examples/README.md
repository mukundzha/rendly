# Example Images

Place your demo images in this folder.

## Supported formats
- PNG
- JPG/JPEG
- GIF
- WebP
- SVG

## Usage

After adding your images to this folder, reference them in your code:

```
SHOW "/examples/your-image.png" FADE IN FOR 2 SECONDS
```

## Example files to add
- `demo.png` - Main product demo screenshot
- `feature1.png` - Feature highlight
- `screenshot.png` - App screenshot
